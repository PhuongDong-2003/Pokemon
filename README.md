# Pokemon
```bash 
Find Pokemon that share the same technique on Json file and calculate the running time of the program with C# object-oriented programming language
```
# Installation

```bash 
$ clone the repo: git clone https://github.com/PhuongDong-2003/Pokemon
```
# Usage
```bash 
Link to refer to moves in Json on Pokemonapi: https://pokeapi.co/docs/v2#moves-section
Run the CLI command: dotnet run "mega-punch" "fire-punch" "thunder-punch" "scratch" "swords-dance" "cut" "ddd" "wing-attack" "snore" "mega-kick"
After the command 'dotnet run' is the input data of Pokemon's moves
```
